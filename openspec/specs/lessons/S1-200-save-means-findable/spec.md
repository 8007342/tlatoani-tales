# S1-200 — Save means findable

## Abstract

Copy-pasting isn't saving. Saving means you can find it again — by name, by query, by index, by anybody other than yesterday-you holding the right tab.

## Aha moment

Persistence is a *retrieval* property, not a *write* property. A file you can't find is noise you paid disk for. The test isn't "did I store it?" — it's "can future-me get it back in one query?"

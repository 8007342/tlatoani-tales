# Symbol Dictionary

## Purpose

Visual vocabulary shared across strips. Each symbol maps a concrete drawing to an abstract engineering idea. When a symbol appears, it means *that specific thing* — consistency builds literacy over the season.

## Core symbols

| Symbol | Drawing | Means |
|---|---|---|
| Floating speech bubbles | Soft pastel rectangles with small tags ("Prompt v17", "Rules", "Example") | Prompts, context fragments, in-flight messages |
| Overstuffed box | Cardboard crate labeled "CONTEXT" with a percentage | Context window pressure |
| Explosion of notes | Bubbles flying outward in a burst | Volatile memory failure |
| Umbrella (open) | Tlatoāni's umbrella deflecting debris | Protection from chaos / boundary |
| Notebook (open, written) | Tlatoāni's notebook with clean hand-written lines | Durable truth / committed memory |
| Treadmill | Covi running on a treadmill, going nowhere | Motion mistaken for progress |
| Maze | Covi in a tangle of corridors | Blind iteration |
| Staircase | Steps climbing upward, each tagged | Incremental convergence |
| Blueprint | Rolled parchment with neat drawings | Specification / target state |
| Breadcrumb trail | Dotted trail of small marks | Telemetry |
| Numbered notebook | Pages labeled 1, 2, 3 ... | Ordered events / Lamport-like truth |
| Hourglass | Glass with sand; two hourglasses side-by-side compare | Time ordering *(TT #04)* |
| Scroll with seal | Sealed parchment | Spec / contract *(TT #05)* |
| Telescope | Tlatoāni's telescope pointed at a distant star | Observability beyond the immediate |
| Lantern held high | Lantern illuminating a dark room | Clarity, signal |

## New symbols

Introducing a new symbol requires:
1. Entry in this spec with: drawing description, meaning, first-appearance strip.
2. A single concrete render in `characters/symbols/<name>.png` used as reference.
3. Trace annotation in the introducing strip's proposal.

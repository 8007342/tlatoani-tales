# Calmecac web bundle

Static viewer served by the httpd container described in
`openspec/specs/calmecac/spec.md`. No build step — every artefact in this
directory is shipped as-is.

`@trace spec:calmecac`
`@Lesson S1-1000`

## Production path

The production path is **not** this README. The bundle is mounted read-only
into an industry-standard tiny httpd container per the Calmecac spec. See
`scripts/tlatoāni_tales.sh` for the Silverblue-only local launcher, and
`images/calmecac/Containerfile` for the image.

## Developer-only local iteration (no container)

You can preview the bundle without the container when iterating on the
viewer itself. This path is a developer convenience, not the production
path.

```bash
cd calmecac/web
python3 -m http.server 8088
# then open http://localhost:8088/
```

If `calmecac-index.json` is not present at the bundle root, the page falls
back to `examples/calmecac-index.json` so views render with sample data.

The production index is generated by `tt-calmecac-indexer` (Rust) and
written to the same directory at build time. The bundle does not care
which tool wrote the index — it only reads the file.

## Files

| Path | Role |
|---|---|
| `index.html` | Shell; references styles and the ES-module entry point. |
| `manifest.webmanifest` | PWA manifest; installable, standalone. |
| `service-worker.js` | Cache-first for static assets and the index. |
| `styles/` | Hand-written CSS; no framework. |
| `scripts/main.js` | Entry; loads index, wires router, registers worker. |
| `scripts/router.js` | Hash-based router. |
| `scripts/breadcrumbs.js` | Breadcrumb trail renderer. |
| `scripts/abstraction-lint.js` | Runtime assertion — no substrate vocabulary leaks into UI text. |
| `scripts/views/` | One small module per view. Each exports `render(container, ctx, params)`. |
| `assets/icons/` | SVG icons used by the manifest and favicon. |
| `assets/fonts/` | Empty; fonts load from the archive's asset path if bundled. |
| `examples/calmecac-index.json` | Fallback sample index for offline iteration. |

## Hard constraints

- Vanilla JS, ES modules, no transpilation.
- No CSS framework; minimal hand-written CSS.
- No substrate vocabulary in UI copy. The abstraction lint enforces this at
  runtime in addition to the build-time checks in `tt-calmecac-indexer`.
- All user-facing copy uses the macron form **Tlatoāni**.
- External archive links are the only substrate-touching action, gated
  behind an explicit "Open on archive" click.
